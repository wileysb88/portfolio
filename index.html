So You Think You Can Style?

An interactive game in which players pick the best outfit for the needs of a client. Run with HTML, CSS, Javascript and jQuery, this game utilizes drag and drop features to navigate through a list of stories, each with their own styling needs. It is able to be applied to any retailer and could be easily used to train staff members on new products each season.

Chomp!

A word of mouth forum that allows locals to share stats and reviews about neighborhood restaurants. Using Ruby, Sinatra, SQLite, HTML and CSS, visitors can vote reviews up or down and live chat to discuss the best local neighborhood spots.

WHO/WHAT/WHERE

Whether you want to snap a picture and post it to the corresponding location, or you are sitting at home wondering what is going on in your city, this site allows users to post pictures and comments to corresponding locations in order to create a crowd sourced live map of what is happening and where using HTML, CSS, React.js and MongoDB.
